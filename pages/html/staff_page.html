About

This system was designed to improve paper based registration systems by re-innovating the process using technology. By doing this, an improved link can be established between its users and the authorities monitoring their activity.

To summarise, DOYRMS registration allows users to sign in and out of a location using a digital kiosk. By being digital, staff users are able to see where users are located at all times, as well as further functionality through a designated search area which allows staff to search for specific users. By defining events, administrators can set quotas that must be met by users. If a user fails to sign in or out for an event, staff users will be notified through the web interface.
